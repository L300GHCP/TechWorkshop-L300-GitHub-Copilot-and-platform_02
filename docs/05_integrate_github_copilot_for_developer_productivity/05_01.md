# Task 01 - Using GitHub Copilot to create Issues (Example)

## Introduction

Zava's Storefront solution has been upgraded from .NET 6 to .NET 8 LTS. This task demonstrates how GitHub Copilot was used to create an issue for this upgrade, showcasing how Copilot can help with creating issues and automating updates to the latest .NET LTS version.

This section provides an example of the process used to create an issue for the .NET upgrade that was then implemented.

## Description

This task demonstrates how an issue was created to describe the requirements to update the solution from .NET 6 to the latest LTS version of .NET (which has now been completed).

## Learning Objective

- Understand how to use GitHub Copilot in Visual Studio Code to create issues.

## Learning Resources

- [Using GitHub Copilot to create or update issues](https://docs.github.com/en/copilot/how-tos/use-copilot-for-common-tasks/use-copilot-to-create-or-update-issues){:target="_blank"}

## Key Tasks

### 01: Open Chat with Copilot

Open GitHub Copilot Chat from your repository to create the .NET upgrade issue.

<details markdown="block">
<summary><strong>Expand this section for detailed steps</strong></summary>

1. Navigate to your forked repository for this lab at GitHub and select Chat with Copilot from the top menu:

    ![Chat with Copilot Icon](../../media/0201_chatcopilot.png)

2. Confirm that you have the correct repository context. If you started from the repository from Chat, you should see it selected:

    ![Chat with Copilot dialog](../../media/0201_chatcopilot_2.png)

</details>

### 02: Create the issue

Create a GitHub issue using Copilot Chat that describes the .NET upgrade from .NET 6 to .NET LTS. 

<details markdown="block">
<summary><strong>Expand this section for detailed steps</strong></summary>

1. From Chat with Copilot enter the following prompt to create an issue:

    ```bash
    /create-issue Update .NET to the current LTS version
    ```
    ![Create an issue with Copilot](../../media/0501_create_issue.png)

2. The end result should be that GitHub Copilot creates an issue that looks similar to the following:

    ![New issue](../../media/0501_new_issue.png)
    
3. Click Create to create the issue.

4. Click on the issue number to open the issue you just created.

    ![Open the issue](../../media/0501_open_issue.png)

</details>

## Summary

This example demonstrated how GitHub Copilot can generate issues that describe upgrading solutions from .NET 6 to .NET LTS. The Zava Storefront has been upgraded to .NET 8 LTS.
